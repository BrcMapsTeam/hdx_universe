This is a network map of the 6000+ dataset on the Humanitarian Data Exchange (HDX). The network is create by looking at the overlap in tags between data sets. There were over half a million links between different datasets. Here is just a sample of the strongest links which give a feel for the structure of the data relationships in HDX.

Use the mouse and controls to explore the map and see the clusters. The play button can be used to auto explore the map.

Unfortunately, it is advisable to not view on a mobile device
